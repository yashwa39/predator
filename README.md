# PREDATOR Website

A modern, professional website for PREDATOR (Predictive Real-time Emergency Detection and Alert Response) - an innovative hazard detection and alert system.

## Features

### Design & User Experience
- **Modern Safety Theme**: Bold red and blue color scheme with emergency response aesthetics
- **Responsive Design**: Fully responsive layout that works on all devices
- **Smooth Animations**: Engaging animations and transitions throughout
- **Interactive Elements**: Hover effects, animated counters, and interactive demos
- **Accessibility**: Keyboard navigation, focus indicators, and screen reader support

### Sections
1. **Hero Section**: Eye-catching headline with animated background and call-to-action buttons
2. **About Section**: Overview of PREDATOR with problem-solution cards and process diagram
3. **How It Works**: Interactive tabs showing scientific foundation, workflow, and sensor demo
4. **Features**: 8 key features with icons and descriptions
5. **Technology Stack**: Subject integration and technical details
6. **Team Section**: 11 team roles with professional layout
7. **Impact & Vision**: Animated statistics and testimonials
8. **Contact Form**: Fully functional form with validation
9. **Footer**: Quick links and social media integration

### Interactive Features
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Tab System**: Interactive tabs in the "How It Works" section
- **Sensor Demo**: Simulated colorimetric gas sensor demonstration
- **Animated Statistics**: Counters that animate when scrolled into view
- **Form Validation**: Real-time form validation with success/error messages
- **Smooth Scrolling**: Smooth navigation between sections

### Technical Implementation
- **HTML5**: Semantic markup with proper structure
- **CSS3**: Modern styling with Flexbox, Grid, and animations
- **Vanilla JavaScript**: No external dependencies, pure JavaScript
- **Performance Optimized**: Throttled scroll events and efficient animations
- **Cross-browser Compatible**: Works on all modern browsers

## File Structure
```
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations

```

## Usage
Simply open `index.html` in any modern web browser. No server setup required.

## Browser Support
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## Performance
- Fast loading times with optimized CSS and JavaScript
- Smooth 60fps animations
- Mobile-optimized for touch interactions
- Accessibility compliant (WCAG 2.1)

## Customization
The website is built with modular CSS and JavaScript, making it easy to customize:
- Colors can be changed in the CSS variables
- Content can be updated in the HTML
- Animations can be modified in the JavaScript file
